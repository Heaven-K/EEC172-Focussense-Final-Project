Project Overview

FocusSense collects environmental data from a light sensor and temperature sensor. The CC3200 processes the readings and adjusts the LED lighting system to maintain comfortable working conditions for students and engineers.

Key Features
FocusSense creates a smarter workspace by sensing environmental conditions and automatically adapting the lighting system. By combining real-time sensor data with embedded control, the system helps maintain a comfortable and productive environment.

System Architecture
Sensors communicate with the CC3200 over I2C. The microcontroller computes the lighting profile and updates both the WS2812B LED strip and the OLED user interface.
System architecture showing sensor inputs, CC3200 processing, and LED/OLED output modules.
System Components
  • CC3200 LaunchPad Microcontroller
  • BH1750 Light Sensor (I2C)
  • TMP102 Temperature Sensor (I2C)
  • WS2812B Addressable LED Strip
  • SSD1306 OLED Display (I2C)
